# Synthesis and Characterisation of Cu(II) and Ni(II) Schiff Base Complexes

## Project Context
This was my MSc Chemistry research project, focused on the synthesis and characterisation of Cu(II) and Ni(II) Schiff base metal complexes and the interaction of the Cu(II) complex with human serum albumin (HSA).

## Research Aim
To synthesise tridentate N,O,N donor Schiff base metal complexes, characterise their structural and physicochemical properties, and study the interaction of the Cu(II) complex with HSA using absorption spectroscopy.

## Methods Used
- Inorganic synthesis using reflux and controlled reaction conditions
- UV–Vis spectroscopy
- Cyclic voltammetry
- X-ray diffraction interpretation
- Elemental composition comparison
- HSA binding analysis using absorption spectroscopy

## Key Findings
- Cu(II) and Ni(II) complexes were synthesised using a tridentate N,O,N donor Schiff base ligand.
- Spectroscopic and analytical results supported metal–ligand coordination through imine nitrogen and phenolate oxygen donor sites.
- Cyclic voltammetry of the Cu(II) complex showed quasi-reversible redox behaviour.
- HSA binding was investigated using concentration-dependent absorption spectroscopy.
